

\# Encrypted Assistant



Encrypted Assistant is an AI model trained to answer basic questions about Zama.

It uses Zama to encrypt the communication on-chain.



URL: https://encryptedassistant.xyz




\## Usage



Install dependencies:

```

npm install

```



Run development server:

```

npm run start

```



Build for production:

```

npm run build

```



Run tests:

```

npm run test

```
